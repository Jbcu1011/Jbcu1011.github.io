Max Temperature Humidity Cloudiness Wind Speed
Summary: Latitude vs. X

included: temperature, cloudiness, wind speed, data and visualizations created as part of the descriptions of any trends and correlations

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. latitude. Factors we looked at and humidity. This site provides the source analysis, as well as explanations and witnessed.
